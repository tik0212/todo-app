# To-Do Web Application

Адаптивное веб-приложение для управления задачами, написанное на чистом JavaScript.

## 🚀 Стек технологий
* **JavaScript (ES6+)** — логика приложения, работа с DOM и событиями.
* **HTML5/CSS3** — семантическая разметка и стилизация (Flexbox).
* **LocalStorage** — сохранение данных на стороне пользователя.
* **Git** — контроль версий.

## ✨ Основные функции
* **Полный CRUD:** создание, редактирование (double-click), выполнение и удаление задач.
* **Фильтрация:** просмотр всех, активных или завершенных дел.
* **Автосохранение:** список задач восстанавливается после перезагрузки страницы.
* **Адаптивность:** интерфейс оптимизирован под мобильные устройства.
